# Atlas Docs Sample

This folder contains a minimal sample of `atlas_docs` files for Atlas Command / Dipsy FAQ Brain.

Files:

- `load.md` — definition of a Load
- `load_statuses.md` — canonical Load status definitions
- `billing_workflow.md` — high-level Billing Workflow

Each file uses YAML frontmatter with:

- `title`
- `domain`
- `doc_type`
- `summary`
- `version`
- `last_updated`
- `related_docs`

You can use these to test your import pipeline into the `atlas_docs` table in Supabase.
